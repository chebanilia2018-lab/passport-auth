# Passport Auth

Учебный проект по авторизации на Node.js + Express.

Сделан в рамках занятия, чтобы разобраться как работает логин/регистрация через Passport и сессии.

---

## Что здесь есть

В проекте реализовано:
- регистрация пользователя (email + пароль)
- вход в систему через Passport Local Strategy
- защищённая страница, которая открывается только после логина
- выход из аккаунта

Пароли сохраняются в зашифрованном виде (bcrypt), а вход держится через сессии.

---

## Маршруты

- `/register` — регистрация
- `/login` — вход
- `/protected` — защищённая страница (только для авторизованных)
- `/logout` — выход из системы

---

## Как запустить

Установить зависимости:
npm install

Запустить сервер:
npm run dev

Создать файл `.env`:
SECRET_KEY=your_secret_key

---

## Используемые технологии

Node.js, Express, Passport.js, express-session, bcrypt, dotenv
